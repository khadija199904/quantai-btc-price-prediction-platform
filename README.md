##  Overview
This project is an end-to-end predictive platform designed to forecast the Bitcoin price at T+10 minutes using real-time market data from the Binance API.

It covers the full data value chain: data ingestion, feature engineering, machine learning model training, and exposure of predictions through a secure REST API.

##  Business Context
In the Fintech and hedge fund industry, real-time data processing and short-term market prediction provide a major competitive advantage.

Quant-AI aims to validate a predictive prototype capable of transforming raw crypto market data into actionable insights using a scalable and automated architecture.

##  Key Features
- Real-time OHLC & volume ingestion from Binance API
- Technical indicators computation (RSI, MACD, EMA, etc.)
- Machine Learning model for short-term price prediction (T+10 min)
- Automated ETL & ML pipelines
- REST API for prediction exposure
- Secure access & monitoring-ready architecture
